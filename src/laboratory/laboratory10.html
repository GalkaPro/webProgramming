Цель работы

Изучить возможности среды для создания одностраничного приложения на React - Create React App.

Ход работы

  1. Изучите теоретические сведения в лекции 10.
  2. Создайте новое React-приложение посредством create-react- app:

    Для разработки понадобятся Node.js не ниже версии 8.10 и npm не ниже версии 5.6.

    Выполните следующую команду 

    npx create-react-app my-app

    Для запуска приложения выполните следующие команды: 

    
                cd my-app
                npm start

    Тут мы переходим в папку проекта и запускаем сервер разработки, который позволяет открыть новое React-приложение, перейдя в браузере по адресу http://localhost:3000/.

    *Создание нового react-приложения в ОС Windows:

    Устанавливаем на свой компьютер node и npm перейдя по ссылке https://nodejs.org/en/

    Перезагрузите компьютер. Затем устанавливаем глобально сам create-react-app:

    npm install -g create-react-app

    Проверяем:

    create-react-app --version

    Если видите версию, то create-react-app успешно установился и можно начинать работать).

    Затем, для создания шаблона приложения, выполните такую команду:

    create-react-app my-app

    Для запуска приложения выполните следующие команды:

                    
                    cd react-intro (переходим в папку где лежит проект)
                    npm start (запускаем проект)
  3. Создайте функциональный компонент MyInfo, формирующий следующие HTML-элементы и выведите экземпляр компонента MyInfo на веб-страницу.:
    1. Элемент <h1> с вашим именем.
    2. Текстовый абзац (элемент <p>), содержащий ваш краткий рассказ о себе.
    3. Список, маркированный (<ul>) или нумерованный (<ol>), с перечнем трёх мест, которые вам хотелось бы посетить.

    Для начала нужно импортировать в файл необходимые библиотеки: react и react-dom. 

                    
                    import React from "react"
                    import ReactDOM from "react-dom"

    После этого надо воспользоваться методом render() объекта ReactDOM, передав ему описание элемента, который мы хотим вывести на страницу и указав место на странице, куда должен быть выведен этот элемент. 

                    
                    function MyInfo() {

                    }

                    ReactDOM.render(
                    <MyInfo />,
                    document.getElementById("root")
                    )

    Теперь нужно вернуть из MyInfo JSX-код, формирующий HTML-разметку в соответствии с заданием. 

                    
                    <div>
                      <h1>Bob Ziroll</h1>
                      <p>This is a paragraph about me...</p>
                      <ul>
                        <li>Thailand</li>
                        <li>Japan</li>
                        <li>Nordic Countries</li>
                      </ul>
                    </div>

    Вот полный код MyInfo.js 

                    
                    import React from "react"
                    import ReactDOM from "react-dom"

                    function MyInfo() {
                        <div>
                      <h1>Bob Ziroll</h1>
                      <p>This is a paragraph about me...</p>
                      <ul>
                        <li>Thailand</li>
                        <li>Japan</li>
                        <li>Nordic Countries</li>
                      </ul>
                    </div>
                    }

                    ReactDOM.render(
                    <MyInfo />,
                    document.getElementById("root")
                    )
  4. Стилизуйте элементы страницы, самостоятельно узнав о том, как это сделать (поищите в Google). Более подробно о стилизации компонентов в React мы ещё поговорим на лекциях.
  5. Сохраните созданные изменения на гитхаб, отправьте в оригинальный репозиторий pull request, аналогично тому, как вы это делали в работе №8.
  6. Оформите отчет о проделанной работе и сдайте его в гуглкласс. Отчет должен содержать: титульный лист, цель работы, теоретические сведения, ход работы, полученные результаты, выводы.
